## sentiment analysis by twitter API 

sentiment analysisは、twitter API を用いて感情分析するプログラムです。

### Installation
#### Dependencies

Apache >= 2.2
PHP >= 5.6.
mysql >= 5.6
composer >= 1.0

#### Modules and API
・mecab
https://taku910.github.io/mecab/#install-unix
